Willkommen bei LehrGrapht.

Ein kostenfreies Word AddIn zum Erstellen von maßstabsgetreuen Plots für Lehrkräfte.

Download (Microsoft Marketplace) GitHub (Source Code)

Plattformen

Verfügbar für Microsoft Word auf Windows, macOS und iPad.

Maßstabsgetreue Plots

Die vom Plotter erzeugten Graphen sind maßstabsgetreu auf 5x5 mm Karopapier.

Funktionen

Beliebig viele Schaubilder pro Word-Dokument. Beliebig viele Graphen und Punkte pro Schaubild in unterschiedlichen Farben.

Plot Liste

LehrGrapht zeigt Dir eine Liste aller im Dokument erkannter Plots an, sodass Du diese editieren und anpassen kannst.

Plot-Editor

Der Plot-Editor erlaubt Dir, einen neuen Plot im Word-Dokument einzufügen. Du kannst hierbei beliebig viele Graphen pro Schaubild erzeugen oder auch Punkte einzeichnen lassen. Der Wertebereich gibt an, in welchen Grenzen die Graphen/Punkte angezeigt werden.

Mehrere Graphen und Punkte pro Schaubild

LehrGrapht kann pro Schaubild mehrere Graphen anzeigen und Punkte einzeichnen. Es obliegt Dir, ob Du nur Graphen, nur Punkte oder beides gemischt anzeigen möchtest.

Word for Web

LehrGrapht wurde für die Desktop-Versionen von Word entwickelt, die auf Windows, macOS oder iPad laufen. Du kannst LehrGrapht auch in Word for Web benutzen, allerdings mit einer Einschränkung: Wenn Du die Textumbruch-Option veränderst und "Hinter dem Text" oder "Vor dem Text" nutzt, dann wird LehrGrapht den Plot nicht mehr zum Editieren erkennen können. Du siehst den Plot dann auch nicht mehr in der Liste aller Plots im Dokument.

Wenn Du allerdings die Option wieder umstellst, kann LehrGrapht den Plot erkennen und editieren. Alternativ kannst Du natürlich den Plot mit einer anderen Layout-Option setzen, bspw. mit einer Tabelle.

Das ist leider eine technische Einschränkung seitens Microsoft. Sobald es sich ändert, wird LehrGrapht dahingehend verbessert.

Häufig gestellte Fragen

Welche Word-Versionen werden von LehrGrapht unterstützt?
LehrGrapht funktioniert auf allen Desktop-Varianten von Word (macOS, Windows) und iPad. LehrGrapht funktioniert eingeschränkt in der Web-Version von Word.
Was unterscheidet LehrGraph von anderen Plottern, bspw. GeoGebra?
LehrGrapht ist maßstabsgetreu und generiert Plots für 5x5 mm Karopapier, dass von den Schüler:innen nachgemessen werden kann.
Was kostet LehrGrapht?
LehrGrapht ist kostenfrei und sogar Open Source auf GitHub! Der Entwickler freut sich jedoch über eine nette E-Mail (support@lehrgrapht.de), eine kleine Spende via Ko-Fi oder einfach über das Weitererzählen an Deine Kolleg:innen!
Was ist das für ein kryptischer Text im Feld "Alternativtext"?
Im Feld "Alternativtext" der Plots siehst einen für Dich kryptischen Text wie plot-7078a878-7039-4ddf-bcde-1089fc56b106. Das ist eine Identifikationsnummer für LehrGrapht, um die Einstellungen für den Plot wieder zu finden, sodass Du ihn editieren kannst. Du solltest dieses Feld am besten nicht verändern, sonst kann LehrGrapht die Einstellungen zum Plot nicht mehr finden.
Warum kann ich den Plot nicht anklicken und ihn dann direkt editieren?
Das ist leider eine technische Einschränkung seitens Microsoft. Ein Add-In kann nicht erkennen, ob Du einen Plot selektiert hast. Daher kann Dir LehrGrapht den Plot nicht direkt zum Editieren anbieten. Du kannst aber jederzeit die Liste der Plots benutzen, um den Plot zu editieren. Sobald es sich ändert, wird LehrGrapht dahingehend verbessert.
Sammelt LehrGrapht Daten?
Nein. LehrGrapht sammelt keinerlei Daten. Da LehrGrapht Open Source ist, kannst Du dies auch jederzeit selbst nachprüfen. Du findest den Source Code auf GitHub.
Ich hab ein Problem, wohin kann ich mich wenden?
Du kannst bei Problemen entweder direkt einen Fehler bei GitHub melden oder Du schreibst dem Entwickler eine E-Mail: support@lehrgrapht.de
Mir fehlt noch eine Funktion in LehrGrapht, was kann ich tun?
Falls Du noch Ideen für weitere Features in LehrGrapht hast, die Du gerne hättest, kannst Du sie entweder auf GitHub vorschlagen oder Du schreibst dem Entwickler eine E-Mail: support@lehrgrapht.de
