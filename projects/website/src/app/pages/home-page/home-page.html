Willkommen bei LehrGrapht.

Ein Word AddIn zum Erstellen von maßstabsgetreuen Plots für Lehrkräfte.

Plattformen

Verfügbar für Microsoft Word auf Windows, macOS und iPad.

Maßstabsgetreue Plots

Die vom Plotter erzeugten Graphen sind maßstabsgetreu auf 5x5 mm Karopapier.

Funktionen

Beliebig viele Schaubilder pro Word-Dokument. Beliebig viele Graphen und Punkte pro Schaubild in unterschiedlichen Farben.

Plot Liste

LehrGrapht zeigt Dir eine Liste aller im Dokument erkannter Plots an, sodass Du diese editieren und anpassen kannst.

Plot-Editor

Der Plot-Editor erlaubt Dir, einen neuen Plot im Word-Dokument einzufügen. Du kannst hierbei beliebig viele Graphen pro Schaubild erzeugen oder auch Punkte einzeichnen lassen. Der Wertebereich gibt an, in welchen Grenzen die Graphen/Punkte angezeigt werden.

Mehrere Graphen und Punkte pro Schaubild

LehrGrapht kann pro Schaubild mehrere Graphen anzeigen und Punkte einzeichnen. Es obliegt Dir, ob Du nur Graphen, nur Punkte oder beides gemischt anzeigen möchtest.

Häufig gestellte Fragen

Welche Word-Versionen werden von LehrGrapht unterstützt?
LehrGrapht funktioniert auf allen Desktop-Varianten von Word (macOS, Windows) und iPad. LehrGrapht funktioniert nicht in der Web-Version von Word.
Was unterscheidet LehrGraph von anderen Plottern, bspw. GeoGebra?
LehrGrapht ist maßstabsgetreu und generiert Plots für 5x5 mm Karopapier, dass von den Schüler:innen nachgemessen werden kann.
Was kostet LehrGrapht?
LehrGrapht ist kostenfrei und sogar Open Source auf GitHub! Der Entwickler freut sich jedoch über eine nette E-Mail (support@lehrgrapht.de), eine kleine Spende via Ko-Fi oder einfach über das Weitererzählen an Deine Kolleg:innen!
Ich hab ein Problem, wohin kann ich mich wenden?
Du kannst bei Problemen entweder direkt einen Fehler bei GitHub melden oder Du schreibst dem Entwickler eine E-Mail: support@lehrgrapht.de
